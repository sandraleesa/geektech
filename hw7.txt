1. Структуры данных – это такие структуры, 
которые могут хранить некоторые данные вместе.
2. Они нужны для хранения связанных данных.
3. Hash Map - это, как я поняла, те же словари, т.е. данные в виде "ключ-значение"
4. LinkedList - это связный список, который создает последовательность из каких-либо элементов.
5. Linked List используем, когда нам нужно создать последовательность из неограниченного количества элементов без доступа к произвольному элементу по индексу.
6.Hash Map используем для легкого и быстрого извлечения данных с помощью hash function.
